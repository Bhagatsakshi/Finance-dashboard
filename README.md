# Finance Dashboard

## Overview

This is a simple finance dashboard built using React and Bootstrap.
It helps users track their financial data, view summaries, and manage transactions in an easy and interactive way.

---

## Features

* Dashboard with Balance, Income, and Expense summary
* Line chart for financial trends
* Bar chart for income vs expense comparison
* Transactions list with details
* Add, Edit, and Delete transactions (Admin only)
* Search/filter transactions
* Role-based UI (Viewer / Admin)
* Insights section (e.g., highest spending)
* Dark mode support
* Data stored using localStorage
* Responsive design for different screen sizes

---

## Tech Stack

* React.js (JavaScript)
* Bootstrap (Styling)
* Chart.js (Graphs)

---

## How to Run

1. Install dependencies:

```
npm install
```

2. Start the app:

```
npm start
```

3. Open in browser:

```
http://localhost:3000
```

---

## How It Works

* Viewer role can only view data
* Admin role can add, edit, and delete transactions
* Data is saved in localStorage, so it stays after refresh
